WASD to control the movement of the camera, hold shift to go slightly faster. 
1-4 are fixed cameras, space to release back into a free camera. 
G to apply sepia post processing effect
B to apply blur effect
P to remove all effects

Normal mapping has been added to the table since the demonstration on Friday. 
Table has extra directional_light on it in order to display normal mapping correctly as it was too dark to see before.
